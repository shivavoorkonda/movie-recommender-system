How the Hybrid Engine Works

🤝
Collaborative Filtering

Matrix Factorisation via SGD. Learns user & movie latent factors. Predicts ratings based on similar users' behaviour.

R ≈ P · Qᵀ + biases
+
🎯
Content-Based

TF-IDF vectorises genre strings. Cosine similarity finds movies with matching feature profiles.

sim(A,B) = A·B / (|A||B|)
→
⚡
Hybrid Fusion

Weighted blend: α·CF + (1-α)·CB. Alpha auto-reduces for new users (cold-start).

default α = 0.7

About This Project

A full-stack hybrid movie recommender system built from scratch

🧠

Machine Learning

Custom Matrix Factorisation trained via Stochastic Gradient Descent — the same maths behind Netflix's winning approach in the Netflix Prize. No external ML libraries for the core algorithm (pure NumPy).

NumPy SGDTF-IDFCosine Simscikit-learn
🗄️

Dataset

MovieLens 100K from GroupLens Research — 100,000 ratings from 943 users on 1,682 movies. Matrix sparsity: 93.7%. Auto-downloaded on first run.

MovieLens 100KGroupLens943 Users1682 Movies
⚙️

Backend

Flask REST API serving real-time recommendations from pre-trained models. Endpoints for hybrid, CF, CB, search, similar movies, user history, and statistics.

FlaskPython 3.14joblibREST API
🎨

Frontend

Responsive single-page application with dark mode, real-time search, interactive charts, alpha blending slider, and animated movie cards.

Vanilla JSCSS3ResponsiveSPA
