Ridge Meta-Stacker & MMR

Ridge regression blends predictions. MMR re-ranks top candidates to guarantee diversity.

MMR(i) = λ·Rel(i) - (1-λ)·Sim(i,S)

About This Project

A portfolio-grade advanced hybrid recommender system built from scratch

🧠

Collaborative Ensembling

Ensembles two models: SGD Matrix Factorisation and Alternating Least Squares (ALS). ALS provides numerical stability on sparse matrices while SGD learns biases dynamically, resulting in an accurate combined CF rating prediction.

NumPy SGDSciPy ALSRidge StackingMeta-learning
⚖️

Fairness & Auditing

Contains self-auditing tools to evaluate demographic parity (comparing RMSE across gender and age cohorts), check popularity biases (Gini coefficient calculations), and check calibration error rates to prevent model bias.

Gini InequalityDemographic RMSECalibration ErrorNovelty Score

Flask Backend

Exposes robust Flask API endpoints featuring LRU cache mechanisms, side-by-side model comparison serialization, prediction confidence intervals, and cached fairness statistics.
